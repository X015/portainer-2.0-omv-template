# portainer-2.0-omv-template
Portainer 2.0 Application Template

# Instructions

You should be using OpenMediaVault as your OS on your NAS.  
Add the following Shares to your OMV  

- appdata
- books
- comics
- downloads
- movies
- music
- playlists
- podcasts
- tv
